# 🎬 StoryWeaver — The Multimodal Director

### *Direct stories. Don’t just write them.*

StoryWeaver is an AI-powered creative director that transforms a single prompt into a fully orchestrated, interleaved multimedia experience.

Instead of generating plain text, StoryWeaver streams a cohesive narrative enriched with visuals, audio, and cinematic moments — all flowing together in real time.

---

## ✨ Overview

StoryWeaver moves beyond traditional text generation. It allows users to **direct** a story using creative controls like visual style, tone, and pacing — while the AI orchestrates text, imagery, music, and motion into a unified experience.

Think of it as:

* 🧠 A writer’s room
* 🎨 A storyboard artist
* 🎼 A composer
* 🎬 A video editor

All powered by multimodal AI.

---

## 🚀 Features

### 🎥 Director Mode

Control your story with:

* **Visual Style** (e.g., Cyberpunk Noir, Watercolor Fantasy)
* **Narrative Tone** (Whimsical, Suspenseful, Dark, Epic)
* **Pacing** (Slow burn, Cinematic, Fast-cut action)

### 🌊 Real-Time Streaming Narrative

* Text fades in dynamically
* Images appear inline as scenes unfold
* Video clips play seamlessly at key moments
* Media is semantically aligned with the narrative

### 🧠 Multimodal Orchestration

* Interleaved generation of text + media
* Structured prompts for coherent storytelling
* Intelligent scene segmentation

---

## 🏗️ Architecture

```
User Prompt
     ↓
Director Controls (Style / Tone / Pacing)
     ↓
Multimodal AI Orchestration Layer
     ↓
Interleaved Output Stream
     ↓
Cinematic React Frontend Renderer
```

### Core Components

* **AI Brain** – Handles multimodal generation and orchestration
* **Streaming Layer** – Parses structured interleaved content
* **Renderer** – Displays text, images, and video in a fluid UI
* **Director Console** – User-facing creative control panel

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/storyweaver.git

# Navigate into the project
cd storyweaver

# Install dependencies
npm install

# Start development server
npm run dev
```

Create a `.env.local` file:

```
GEMINI_API_KEY=your_api_key_here
```

## 🎮 Usage

1. Enter a story concept or prompt.
2. Adjust the Director Controls:

   * Visual Style
   * Narrative Tone
   * Pacing
3. Click **Direct**.
4. Watch your story unfold in real time.
